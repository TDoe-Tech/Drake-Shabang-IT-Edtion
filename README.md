# drake-shabang-it-edition 🖥️

A fun little side project I built a music video visualizer for Drake's Shabang where every tech reference in the lyrics is swapped out for its real equivalent. Shebang becomes `#!/bin/bash`, "some head" becomes `<head>`, "they dess" becomes `SIGKILL`, you get the idea.

Built this for the fun of it. Sometimes you just want to make something stupid and cool.

## what it does

- Lyrics scroll and highlight in sync with the song
- Tech slang is replaced with actual code, CLI commands, git syntax, HTTP errors, regex, and more
- Terminal aesthetic with scanline effect
- Full play/pause and seek controls

## built with

- Vanilla HTML, CSS, and JavaScript
- YouTube IFrame API for audio
- Zero frameworks, zero dependencies, zero regrets
